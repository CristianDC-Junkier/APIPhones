# API Phones - Ayuntamiento de Almonte
Web App desarrollada para la gestión del **listín telefónico del Ayuntamiento de Almonte**. La aplicación permite almacenar, consultar y gestionar los datos de contacto, así como enviar **notificaciones por correo** a los administradores mediante un sistema interno de mensajería.
## Características
- Gestión completa del **listín telefónico** del Ayuntamiento.
- Envío de **notificaciones por correo** a los administradores.
- Sistema de mensajería interna para la administración de notificaciones.
- API estructurada para integrarse con los datos de los diferentes departamentos.
- Optimizada para uso interno del **Ayuntamiento de Almonte**.
